# Roster Redux

Scheduling App using React & Redux. The User can choose which events to attend. The user can choose up to three different events that run simultaneously by allocated time period.

## Tools Used:
* Babel
* ESLint
* React.js
* React Router
* Redux
* Webpack
* Yarn Package Manager
